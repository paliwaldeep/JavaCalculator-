# Java Console Calculator

This is a simple Java-based calculator that runs in the console.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Loop until user chooses to exit

##  How to Run
1. Compile:
   ```bash
   javac Calculator.java
2. Run:  
   java Calculator
